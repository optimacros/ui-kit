## Basic

- сбрасываю пропсы
- проверяю наличие элементов на странице
- проверяю значение (`0`)
- делаю скриншот (`default`)
- кликаю `+` 3 раза 
- проверяю значение (`3`)
- кликаю `-` 5 раз
- проверяю значение (`-2`)

## Step

- сбрасываю пропсы
- передаю пропсы `{ step: 15, maxValue: 1111 }`
- проверяю наличие элементов на странице
- проверяю значение (`0`)
- кликаю `+` 3 раза 
- проверяю значение (`45`)
- кликаю `-` 5 раза 
- проверяю значение (`-30`)
