## Basic

- сбрасываю пропсы
- проверяю наличие элементов

### 1.Max

- нажимаю `create toast` 5 раз
- проверяю количество видимых тостов (5)
- нажимаю `remove all`
- проверяю количество видимых тостов (0)
- передаю пропсы `{ max: 3 }`
- нажимаю `create toast` 5 раз
- проверяю количество видимых тостов (3)
- нажимаю `remove all`
- проверяю количество видимых тостов (0)
